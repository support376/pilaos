# pilaos 콜드 컨택 매뉴얼 v1.0

## 운영팀 SOP

### 원칙

1. **광고가 아닌 비즈니스 제안.** "잠재 매수자 N명이 있다"는 정보 제공이지 매도 권유 아님.
2. **1회 + 90일 쿨다운.** 첫 컨택 후 응답 없으면 90일 후 1회 추가. 그 후 영구 do-not-contact.
3. **거절 의사 즉시 영구 차단.** "더 이상 연락하지 마세요" 단어 시 do-not-contact 리스트 추가, 시스템에서 매물 노출도 함께 hide 검토.
4. **공개된 일반 전화번호로만.** 카카오·네이버 플레이스에 공개된 매장 대표번호. 비공개 정보 사용 금지.

### 메시지 템플릿 (카톡/문자 기준)

```
안녕하세요, [매장명] 원장님.

저는 pilaos 운영팀입니다. 전국 필라테스 영업양수도 매칭을 도와드리는 곳입니다.

[시군구] 권역에서 [매장명]과 비슷한 조건의 매물을 찾는 잠재 매수자 분들이 등록되어 있어 안내차 연락드립니다.

— 우리 추정으로 권리금 [금액], 월매출 [금액] 수준
— 매도 의향이 있으시면 익명 노출로 시장 반응만 가볍게 확인해드립니다
— 매도 의사가 없으시면 답장 안 주셔도 됩니다

매물 정보 잘못된 부분이 있으면 그것만 알려주셔도 정정해드립니다.

매물 페이지: https://pilaos.vercel.app/listings/[ID]
운영팀: support@pilaos.app
```

### 금지 사항

- ❌ 첫 메시지에서 "수익률 13%" 등 단정 표현 (추정임을 명시 필수)
- ❌ "매도 시그널 점수 80점" 외부 노출 (명예훼손 위험)
- ❌ 같은 번호 재발송 30일 내
- ❌ 거절 응답 후 재연락
- ❌ 카톡 친구추가/광고 동의 강요
- ❌ 매장 사진을 첨부 (저작권)

### 응답 분류

| 응답 | 처리 |
|---|---|
| 매도 의향 있음 | 매도 폼 링크 안내 → /sell/new?listing=PIL-... |
| 매도 의향 미정 | 90일 후 follow-up |
| 정보 정정만 요청 | 정정 처리 후 do-not-contact (재연락 X) |
| 명시적 거절 | do-not-contact 영구 등록 |
| 폐업/이전 | 매물 hide + do-not-contact |
| 무응답 | 90일 후 1회 추가 → 그 후 영구 do-not-contact |

### 분쟁 대응

매장 측에서 "왜 내 매장이 등록돼있냐"는 항의 시:
1. 사과 + 잠재매물 모델 안내 (이용약관 링크)
2. 즉시 노출 거부 처리 (24시간 내, 임시 hide 즉시)
3. 본인확인 통화 후 영구 hide
4. 사업자번호·휴대폰 do-not-contact 등록

언론·커뮤니티 노출 시:
1. PR 1차 응답 (24시간) — 잠재매물 모델 + 매장 통제권 보장 설명
2. 운영팀 카톡 채널 24시간 응답
3. 필요시 변호사 자문 후 공식 입장 발표

### 감사 로그

- 모든 컨택 시도·응답·거절을 기록
- 운영팀 인박스에 상태 토글 (완료/거절/보류/폐업)
- 월 1회 do-not-contact 리스트 검토
